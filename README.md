# 🌍 Country Explorer Web App

## 📌 Project Description

Country Explorer is a web application that allows users to explore information about countries using real-time data from a public API. Users can search, filter, and sort countries, and view key details about each country in a simple and interactive interface.

## 🌐 API Used

* REST Countries API
* https://restcountries.com/v3.1/all

## 🚀 Planned Features

* 🔍 Search countries by name
* 🌎 Filter countries by region
* 📊 Sort countries by population or name
* 🏳️ Display country flag
* 🏙️ Show capital city
* 👥 Display population
* 🌐 Show region and subregion
* 💰 Display currency
* 🗣️ Show languages spoken
* 📱 Responsive design for all devices

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Fetch API
* Array Higher Order Functions (filter, map, sort)

## ⚙️ Setup Instructions

1. Clone the repository
2. Open `index.html` in a browser
